Font-Family

Font family refers to the font being used in the HTML document. Example of this:


"Segoe UI, Tahoma, Geneva, Verdana, sans-serif" (which is currently what is being used)


Multi-worded font families required the value to be surrounded with quotation marks in order to work. Example of this:


'Segoe UI'

Web Fonts

Web fonts are fonts that are available on the web for users to use when styling their HTML document. Web fonts are spread into 2 categories. These categories are described below:

Web-Safe Fonts Web Link Fonts
Web-safe fonts are fonts that are default across all browsers therefore they will render no matter the browser or user. For this reason, Web-safe fonts are perfect as fallback fonts when preferred fonts do not render. Web Link Fonts can be obtained via websites that are free-use or paid however that it entirely preference.
Please see the links below for examples:
https://www.cssfontstack.com/ https://fonts.google.com/

@font-face formats

font-face/@font-face allows the developer to link local fonts. The font-face rule is a CSS syntax. There are 4 formats of fonts. Please see the table below:

Font name Font Definition
OTF Open Type Font
TTF TrueType Font
WOFF Web Open Font Format
WOFF2 Web Open Font Format 2

Notice!


Font format order is crucial for browsers as fonts will required time to render. WOFF2, WOFF then TFF is always recommended. WOFF2 uses better compression and therefore renders the optimally on browsers.

Optional Properties

Although these properties are default for all browsers, they can be edited for suitability as well as convenience when text is hard to see/read.


'Letter-spacing' will affect the spacing between letters, which helps with certain fonts and/or styles, takes length values, such as '2px' or '0.5em'.


'Word-spacing' will affect the spacing between words, which helps with reading bold or enlarged text, takes length values as well.


'Line-height' takes length values as well, alternatively, unitless numbers can be used, such as '1.2'.
